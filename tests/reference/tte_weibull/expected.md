# Expected values: TTE Weibull

Dataset: `tte_weibull.csv` — 100 subjects, data-generating `scale = 20`, `shape = 2.0`,
between-subject variability on the **shape** (`omega^2(log shape) = 0.20`), administrative
right-censoring at `t = 30`. Generated by `simulate.R` (seed = 42) → **84 events, 16
censored (16%)**. Scale parameterisation: `H(t) = (t/scale)^shape`.

## Parameterisation note

- **ferx** `[event_model] scale = TVSCALE`, `shape = TVSHAPE*exp(ETA_SHAPE)` reports
  `TVSCALE`, `TVSHAPE` on the **natural** scale; `omega^2 = var(log shape)`.
- **NONMEM** `SCALE = EXP(THETA(1))`, `SHAPE = EXP(THETA(2)+ETA(1))` reports logs. Compare
  `exp(THETA(1))` ↔ ferx `TVSCALE`, `exp(THETA(2))` ↔ ferx `TVSHAPE`.

## Fixed-effects (n_eta = 0) — exact, license-free anchor

`survreg(Surv(TIME,DV)~1, dist="weibull")`, mapped to ferx's `(scale, shape)` via
`shape = 1/scale_sr`, `scale = exp(intercept)` (see `survreg.R`). A ferx fixed-effects
Weibull fit must match it:

This is exactly the model the hand-off **`nonmem.ctl`** ran (`$OMEGA 0 FIX`, `METHOD=0`):

| Quantity | ferx (n_eta=0) | survreg | nlmixr2 (no η) | NONMEM (ω²=0 FIX) |
|---|---|---|---|---|
| scale | 22.1766 | 22.1766 | 22.2064 | 22.173 (θ₁ 3.09887) |
| shape | 2.1192 | 2.1193 | 2.1151 | 2.119 (θ₂ 0.750983) |
| OFV / −2logLik | 640.261 | 640.261 | (OBJF n/c) | 640.261 |

scale agrees to **~0.13%**, shape to **~0.2%**; ferx / survreg / NONMEM −2logLik **identical**
(640.261). Reproduce: `Rscript survreg.R`, `Rscript nlmixr2_fixed.R`.

(The pooled fixed-effects shape 2.12 ≠ the data-generating 2.0 because it collapses the
shape variability — that is what the SSE checks separately.)

## Simulation–estimation (SSE) recovery — truth check, N = 2000

`tte_sse_weibull_recovers_truth`:

| Parameter | Truth | ferx FOCEI (N=2000) | deviation |
|---|---|---|---|
| scale | 20.0 | 19.62 | −2% |
| shape | 2.0 | 2.03 | +2% |
| omega^2 (log-shape) | 0.20 | **0.344** | **+72%** |

**Finding (#440):** FOCEI-Laplace materially **over-estimates** the frailty variance when the
random effect is on a *nonlinear* hazard parameter (here the Weibull **shape**). It does **not**
vanish as the true `omega^2` shrinks (truth 0.05 → ferx ≈ 0.16). A **SAEM** fit of the *same*
data gives `omega^2 ≈ 0.13` — the two estimators straddle the truth (0.20), confirming a FOCEI
*approximation* limitation, **not** a likelihood bug (the fixed-effects fit matches `survreg`
exactly). This matches the plan's guidance that SAEM/IMP are preferred for TTE (§3.3, §13). The
structural parameters (`scale`, `shape`) recover well under both estimators.
