# Expected values: TTE Gompertz

Dataset: `tte_gompertz.csv` — 300-subject, 2-arm RCT, **fixed-effects** Gompertz with a
treatment effect, right-censored at `t = 365`. Data-generating `log_alpha = −6.0`
(`alpha ≈ 0.00248`), `log_gamma = −5.4` (`gamma ≈ 0.00450`), `log_hr = −0.8`. Generated by
`simulate.R` (seed = 42) → **231 events (77%), 69 censored**; events by arm: control 135,
treated 96 (HR < 1, consistent with `log_hr = −0.8`).

`h(t) = alpha·exp(gamma·t)·exp(log_hr·TRT)`,  `H(t) = (alpha/gamma)(exp(gamma·t)−1)·exp(log_hr·TRT)`.

## Parameterisation note

- **ferx** `[event_model] alpha = TVALPHA`, `gamma = TVGAMMA`, `loghr = LHR*TRT` reports
  `TVALPHA`, `TVGAMMA` on the **natural** scale and `LHR` directly.
- **NONMEM** reports `log_alpha = THETA(1)`, `log_gamma = THETA(2)`, `log_hr = THETA(3)`.
  Compare `exp(THETA(1))` ↔ ferx `TVALPHA`, etc.

## Fixed-effects RCT recovery — FOCEI / fixed effects, fitting `tte_gompertz.csv`

This is the `n_eta = 0` path (plan §16 D7) and exercises the `[event_model]` covariate +
`loghr` term. There is no base-R/`survreg` anchor for Gompertz, so recovery of the
data-generating parameters is the guard; NONMEM / nlmixr2 are the cross-tool hand-off.

ferx and nlmixr2 FOCEI recover all three parameters within ~3% on the log scale — strong
license-free agreement (`tte_convergence_gompertz_rct_recovers` / `nlmixr2.R`).

> † **OFV is not cross-tool comparable for Gompertz.** ferx's OFV (3011) is the true −2LL
> (its convention matches `survreg` exactly for Exponential/Weibull); nlmixr2's objective uses
> different normalizing constants (raw OBJF 3562.5, AIC 4119.9) — and unlike Exp/Weibull the
> offset here is large. Compare **parameter estimates**, not OFV, across tools (§11.3).
> nlmixr2 used `est = "focei"` — `bobyqa` (as in the blog) fails to fit this model in nlmixr2 5.x.

## nlmixr2 blog cross-reference (plan §7.6)

The nlmixr2 survival blog (`blog.nlmixr2.org/blog/2026-05-28-survival-nlmixr2/`) fits a
similarly structured 300-patient 2-arm Gompertz RCT (different seed/params) and reports
`log_alpha −6.173`, `log_gamma −5.321`, `log_hr −0.799`, OFV 2955.64. Our dataset differs;
to compare directly, reproduce the blog's dataset and refit with ferx.

## Frailty-Gompertz SSE — nonlinear-frailty omega^2 finding (#440)

`tte_sse_gompertz_recovers_truth` simulates a *frailty* Gompertz (BSV on `gamma`,
truth `alpha=0.002, gamma=0.05, omega^2=0.05`, N=2000) and refits:

| Parameter | Truth | ferx FOCEI | note |
|---|---|---|---|
| alpha | 0.002 | 0.00167 | alpha/gamma collinearity |
| gamma | 0.050 | 0.0541 | +8% |
| omega^2 (log-gamma) | 0.050 | **0.081** | **+62% — FOCEI over-estimates (see #440)** |

Same nonlinear-frailty `omega^2` over-estimation observed for the Weibull shape: FOCEI-Laplace
inflates the frailty variance for a random effect on a nonlinear hazard parameter. Structural
parameters recover; SAEM/IMP are the recommended estimators for these (plan §3.3, §13).
