# Expected values: TTE Exponential

Dataset: `tte_exp.csv` — 100 subjects, data-generating `lambda_pop = 0.1`,
`omega^2 = 0.25` (variance of `log lambda`), administrative right-censoring at
`t = 24`. Generated by `simulate.R` (seed = 42) → **82 events, 18 censored (18%)**.

All tools fit *this same file*, so they should agree **with each other** regardless
of the 0.1 / 0.25 data-generating truth — this particular n=100 realisation happens
to land low (the pooled rate is ~0.075–0.077). Use the large-N SSE (below) to judge
recovery of the *truth*.

## Parameterisation note (read before comparing)

- **ferx** `[event_model] scale = TVLAMBDA*exp(ETA_LAMBDA)` reports `TVLAMBDA` on the
  **natural (rate)** scale; `omega^2 = var(ETA_LAMBDA) = var(log lambda)`.
- **NONMEM** `LAMBDA = EXP(THETA(1)+ETA(1))` reports `THETA(1) = log(lambda)`. Compare
  `exp(THETA(1))` to ferx `TVLAMBDA`, and `OMEGA(1,1)` to ferx `omega^2`.

## Fixed-effects (n_eta = 0) — exact, license-free anchor

For a censored Exponential the MLE is the closed form `lambda = events / sum(time) =
82 / 1100.6`. This is exactly what the ferx fixed-effects (`n_eta = 0`) fit maximises,
so it must match base-R `survival::survreg` to numerical tolerance.

| Quantity | ferx (fixed) | survival::survreg | agreement |
|---|---|---|---|
| lambda (rate) | 0.074506 | 0.074506 | exact (rel err < 1e-4) |
| OFV / −2logLik | 589.888 | 589.888 | exact (≤ 1e-3) |

Reproduce survreg: `Rscript survreg.R` (uses only base-R `survival`).

## Simulation–estimation (SSE) recovery — truth check, N = 2000

Simulating a large dataset from the truth with ferx's own `simulate()` and refitting
drives Monte-Carlo noise to ~0 and isolates estimator bias
(`tte_sse_exponential_recovers_truth`):

| Parameter | Truth | ferx FOCEI (N=2000) | deviation |
|---|---|---|---|
| lambda_pop | 0.100 | 0.099 | −1% |
| omega^2 | 0.250 | 0.232 | −7% |

The ~7% `omega^2` shortfall is the expected **mild FOCEI-Laplace bias for TTE**
(plan §3.3); it worsens as the event rate falls and is reduced by SAEM / IMP.

## Notes

- The exact survreg match (rate *and* −2logLik) confirms ferx's TTE hazard, censoring,
  and likelihood **constants** are correct; the SSE confirms the frailty integration is
  near-unbiased.
- nlmixr2 (`est = "focei"`) is the generalised FOCEI (FD Hessian) path; NONMEM uses
  `METHOD=CONDITIONAL LAPLACIAN INTERACTION`. A small OFV offset between tools is expected
  from differing normalising constants (plan §11.3) — compare parameter point estimates,
  not raw OFV, across tools.
