# Expected values: TTE Weibull

Dataset: `tte_weibull.csv` (100 subjects, true scale=20, shape=2.0, ω²_shape=0.20, right-censored at t=30).
Generated by `simulate.R` (seed=42). Scale parameterization: H(t) = (t/scale)^shape.

## Notes

- Weibull `scale` is a time (AFT interpretation: H=(t/scale)^shape). Covariate on scale = AFT.
- For PH parameterization use `loghr` term on the hazard (plan §3.2).
- NONMEM and ferx should agree within ±0.5 OFV; constant offset arises if one omits the normalizing constant (see plan §11.3).
