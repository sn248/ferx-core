# Expected values: TTE Gompertz

Dataset: `tte_gompertz.csv` (300 subjects, 2-arm RCT, fixed-effects Gompertz, right-censored at t=365).
True params: log_alpha=−6.0, log_gamma=−5.4, log_hr=−0.8.
Generated by `simulate.R` (seed=42).

## Cross-reference: nlmixr2 blog (§7.6 of plan)

The nlmixr2 blog (`blog.nlmixr2.org/blog/2026-05-28-survival-nlmixr2/`) fits a Gompertz model
to a similarly structured dataset (300 patients, 2-arm RCT) and reports:

| Parameter | Estimate | SE |
|---|---|---|
| log_alpha | −6.173 | 0.295 |
| log_gamma | −5.321 | 0.274 |
| log_hr | −0.799 | 0.274 |
| −2LL | 2955.64 | — |
| AIC | 2961.64 | — |

The blog's dataset uses a different seed / true parameters than ours. To use the blog as a
direct comparison, run the blog's inline R code to reproduce its exact dataset, then fit with
ferx and compare to OFV=2955.64. Both are valid validation paths; our dataset below lets us
also verify with NONMEM.

## Notes

- No random effects in this dataset (n_eta=0 fit; pure fixed-effects hazard).
- Validates the `n_eta=0` code path required by plan §16 D7.
- ferx should use `method = bobyqa` or `focei`; Gompertz fixed-effects has no EBE loop.
- Treatment effect enters as `loghr = THETA_LOGHR * TRT` in `[event_model]`.
