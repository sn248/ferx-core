## Notes

- NONMEM uses `METHOD=CONDITIONAL LAPLACIAN INTERACTION` (Laplace approximation).
- nlmixr2 uses `est = "focei"` which is the generalised FOCEI (FD Hessian) path.
- ferx uses FOCEI Laplace (`foce_subject_nll_interaction` + `data_term_hessian_fd`).
- A small OFV offset between tools is expected if they include/exclude normalizing constants differently (see plan §11.3).
- SSE cross-check: `cargo test tte_sse --features survival,slow-tests` should recover true params within 2×SE.
