# Expected values: clock-reset (gap-time) Weibull RTTE

Dataset: `rtte_weibull_reset.csv` — 100 subjects, **clock-reset** (renewal) repeated
time-to-event with a Weibull inter-event-gap distribution and a subject frailty on the
scale: `scale_i = TVSCALE·exp(eta_i)`, data-generating `TVSCALE = 5`, `TVSHAPE = 1.5`,
`ω²(log scale) = 0.09`, administrative censoring at `t = 30`. Generated by `simulate.R`
(seed 32) → **661 events, 6.61/subject**. ferx Weibull parameterisation: `H(t) =
(t/scale)^shape`.

Clock-reset means the hazard clock restarts at each event, so a subject's inter-event
gaps `Δ_k` are independent Weibull draws and the NLL is `Σ_k [H(Δ_k) − log h(Δ_k)]`
(final gap censored). A `shape ≠ 1` gives a time-varying within-gap hazard, so clock-reset
and clock-forward genuinely differ here (they coincide only for the memoryless
exponential).

## Anchors

### 1. Fixed-effects (n_eta = 0) — exact via `survreg` on the gap durations

Without frailty, the reset likelihood reduces **exactly** to an ordinary Weibull survival
regression on the inter-event gaps (observed gaps are events, the final gap is
right-censored). So `survreg(Surv(gap, event) ~ 1, dist="weibull")` (`survreg.R`) is the
closed-form MLE — the gap-time analogue of the standalone `tte_weibull` survreg anchor.

| Source | scale | shape | −2logL |
|---|---|---|---|
| `survreg` on gaps (exact MLE) | 4.78920 | 1.32415 | 3243.856 |
| **ferx** fixed-effects clock-reset | **4.78923** | **1.32415** | **3243.8563** |

### 2. Mixed-effects (frailty) — NONMEM LAPLACE cross-tool

| Source | TVSCALE | TVSHAPE | ω²(log scale) | OFV |
|---|---|---|---|---|
| **NONMEM** LAPLACE (`nonmem.ctl`) | 5.15939 (SE 0.240) | 1.52987 (SE 0.0496) | 0.13225 (SE 0.0241) | 3175.863 |
| **ferx** FOCEI clock-reset | 5.15857 (SE 0.238) | 1.52972 (SE 0.0507) | 0.13228 (SE 0.0289) | 3175.8630 |
| data-generating | 5.0 | 1.5 | 0.09 | — |

ferx reproduces NONMEM LAPLACE to 4–5 significant figures on every parameter and the OFV
to the decimal (3175.863). The frailty ω² sits above the 0.09 generating value on this
single realization (weakly identified), consistently for both tools.

## Notes

- **NONMEM control stream.** Pure-TTE (`$EST … LIKELIHOOD`, no `F_FLAG`). Constant-shape
  Weibull needs no `$DES`: a `TLAST` reset in `$PRED` makes each row contribute the
  Weibull survival/density over its inter-event gap `DEL = TIME − TLAST`. Full output in
  `nonmem.lst` / `nonmem.ext`.
- **Clock-reset ≠ clock-forward here.** Because `shape > 1`, evaluating the hazard on the
  gap (reset) rather than absolute time (forward) changes the likelihood — the unit tests
  in `src/survival/mod.rs` pin both closed forms and assert they differ.

## Reproduce

```
Rscript simulate.R                                   # regenerate rtte_weibull_reset.csv (seed 32)
Rscript survreg.R                                    # exact fixed-effects anchor (base R + survival)
ferx rtte_weibull_reset_fixed.ferx --data rtte_weibull_reset.csv   # ferx fixed  -> survreg
ferx rtte_weibull_reset.ferx       --data rtte_weibull_reset.csv   # ferx mixed  -> NONMEM
# NONMEM (Docker): nmfe76 nonmem.ctl nonmem.lst
```
